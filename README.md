# WatheMixin

一个用Mixin修改 [Wathe](https://github.com/doctor4t/wathe) 的 Fabric 模组。

## 功能

### 移除赞助者限制

> 允许 **[权限等级](https://zh.minecraft.wiki/w/权限等级)4** 使用原本仅限赞助者使用的命令和开发者专属指令

### 开发者专属指令

> `/wathe:lockToSupporters` 指令，原为开发者UUID的专属<br>
> 用于开关赞助者指令的调试指令
